# 위스키 분류 — 비즈니스 요구사항

마스터 기획은 [`../SPEC.md`](../SPEC.md)를 따른다. 공통 흐름·단일 오버레이 상세·세션 전역 상태 원칙은 마스터와 동일하다.

## 목록 화면에 맞는 정보 차원

| 차원 | 설명 |
|------|------|
| 이름 | 제품명 또는 매장 표기명. |
| 카테고리 | 스카치·아이리시·아메리칸(버번·라이 등)·재패니즈 등 넓은 구분. |
| 리전·스타일 | 예: 스카치의 스페이사이드·아일라 등 지역 감, 또는 페이티드(스모키) 여부 등 공통적으로 알려진 특성. |
| 숙성·캐스크 | 예: 셰리 캐스크, 버번 캐스크 등 **대표 표기**(세부는 상세). |
| ABV | 알코올 도수(표기 가능 시). |
| 서빙 단위 | 예: 30ml·45ml·더블 요금 등 매장 규격. |
| 가격 | 선택한 단위 기준가. |

품절·비노출은 마스터 규칙을 따른다.

## 상세 화면 — 표시 항목

위스키 상세는 **향·맛의 여정·마무리**와 **어떻게 마실지**를 분명히 한다.

### 필수

- 이름
- 카테고리 및 주요 지역/스타일 설명(손님 친화 한두 문단)
- ABV
- 가격 및 서빙 단위(샷·더블 등 매장 정의)
- 테이스팅 노트: **향**, **입안**, **피니시**를 구분해 서술(과장 없이 매장 톤 유지)

### 선택

- 추천 드링킹 방식: 스트레이트, 온더록, 하이볼, 소량의 워터 등
- 음용 온도 감(예: 실온·약간 냉각)
- 증류소·표현식(싱글 몰트 등)이 매장 정책에 포함되면 한 줄 소개
- 캐스크 타입·숙성 년수가 메뉴에 중요하면 표시

품절 시 상세에서 주문 불가를 명확히 한다.

## 행동 규칙

- 목록 항목 선택 시 해당 위스키 상세만 연다. 다른 항목 선택 시 단일 상세 교체(마스터).
- 분류가 위스키일 때만 이 필드 세트를 사용한다.

## 예시 메뉴 데이터 (기획용)

실제 제품 설명은 증류소·수입사 **공개 카피·위키 등 일반 자료**와 어긋나지 않게 조정한다. 아래는 스펙 검증용 예시다.

| 이름(예시) | 카테고리·리전 감 | 한 줄 특성 | 상세에 넣을 테이스팅 축(예시) |
|------------|------------------|------------|------------------------------|
| Glenlivet 12yo (예시) | 스카치 싱글 몰트·스페이사이드 | 과일·꽃향이 비교적 부드럽고 입문자에게 무난한 스페이사이드 감. | 향: 사과·배. 입안: 부드러운 당도. 피니시: 짧고 깔끔. |
| Laphroaig 10yo (예시) | 스카치 아일라·페이티드 | 요오드·메디컬한 스모크가 강한 것으로 널리 알려진 아일라 대표 스타일. | 향: 스모크·메디컬. 입안: 짭짤함·피트. 피니시: 길고 스모키. |
| Yamazaki Distiller’s Reserve (예시) | 재패니즈 위스키 | 일본 증류소에서 나오는 과실·미즈나라 오크 힌트가 언급되는 제품군(매장 카피에 맞춤). | 향: 과실·스파이스. 입안: 균형. 피니시: 오크 감 잔존. |
| Buffalo Trace (예시) | 버번·켄터키 | 옥통 숙성 버번으로 알려진 바디감과 바닐라·캐러멜 톤이 흔한 설명. | 향: 바닐라·캐러멜. 입안: 달콤함·스파이스. 피니시: 중간 길이. |
| Redbreast 12yo (예시) | 아이리시 포트 스틸 | 아이리시 싱글 포트 스틸의 부드러운 질감·셰리 캐스크 단말이 자주 언급됨. | 향: 건과일·스파이스. 입안: 크리미. 피니시: 달콤한 여운. |

## 마스터와의 연결

- **현재 분류**가 위스키일 때 목록·상세는 위 차원을 사용한다.
- **열린 상세**가 위스키 항목이면 테이스팅·서빙 제안 필드를 채운다.
